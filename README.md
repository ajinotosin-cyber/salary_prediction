 Salary Prediction App

A machine learning web app that predicts an employee’s monthly salary based on experience, education, and performance.



---



##  Live Demo

https://salary-prediction-app-gyordcdcnkeaudtbbbjnt2.streamlit.app/



---



## Features

- Predicts salary in real-time  

- Simple and interactive UI  

- Uses ML model (Scikit-learn)  

- Handles multiple employee inputs  



---



## Tech Stack

- Python  

- Streamlit  

- Pandas  

- NumPy  

- Scikit-learn
